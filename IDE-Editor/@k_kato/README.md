# VS Code ではじめる Java 開発環境構築入門

Visual Studio Code (VS Code) は軽量なテキスト エディターですが，拡張機能によって Java の IntelliSense (自動補完)，Lint (コード エラーや警告の即時表示) や デバッガーが使えるようになりました。VS Code で Java の開発環境を構築する方法を解説します。
